### Selected work

- **OpenCAD / MyCAD**  
  Building a native parametric 3D CAD application from scratch in C++20.  
  It combines an OCCT geometry kernel, OpenGL rendering, an ImGui/GLFW interface, PlaneGCS sketch constraints, SQLite project storage, timeline-based modeling, live previews, and automation hooks into a compact desktop CAD system.

- **MakerCAD**  
  Designing a simplified CAD shell on top of FreeCAD.  
  MakerCAD keeps FreeCAD's powerful modeling backend, but wraps it in an isolated profile and a cleaner maker-focused UI layer with custom ribbon controls, workflow state handling, browser integration, themes, and session logging.

- **iceball-godot**  
  Rewriting Iceball in Godot 4 as an experiment, then extending it beyond the original idea with custom gameplay and engine-facing features.
